Шаблон для импорта участников мероприятия

Инструкция:

  1. Данные начинаются с 13-й строки
  2. Колонка B - ФИО (ОБЯЗАТЕЛЬНО)
  3. Колонка C - SAP ID (ОБЯЗАТЕЛЬНО для ТП)
  4. Колонка D - Должность (например, "Торговый представитель")
  5. Колонка E - Филиал/Территория
  6. Колонка F - Стаж в днях (365 = 1 год)
  7. Колонка G - Резерв (не используется)
  8. Колонка H - Телефон (в формате +79001234567)
  9. Колонка I - Email (ОБЯЗАТЕЛЬНО для всех пользователей)
A B C D E F G H I
ФИО SAP ID Должность Территория Стаж (дни) Резерв Телефон Email
Пример: Иванов Иван Иванович 12345 Торговый представитель Москва 365 +79001234567 ivanov@sns.ru

Примечания:

  • Пользователи будут созданы в системе, если их еще нет
  • Пароль для всех созданных пользователей: 123456
  • Для идентификации пользователя должен быть указан либо Email, либо SAP ID
  • При указании SAP ID без Email, электронная почта будет автоматически сгенерирована в формате sapID@sns.ru
